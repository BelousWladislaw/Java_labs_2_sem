Конвертер валют
Особенности:
-Выбор двух криптовалют из выпадающих меню.
-Ввод количества первой криптовалюты для конвертации.
-Нажатие кнопки "Рассчитать" для выполнения конверсии.
-Просмотр курса конверсии и сконвертированной суммы в окне результатов.

Использование
1.После запуска приложения появится основное окно.
2.В центральной текстовой области будут отображаться доступные криптовалюты вместе с их символами.
3.Выберите желаемые криптовалюты из выпадающих меню: одну в качестве исходной валюты,
а другую - в качестве целевой.
4.Введите количество исходной валюты для конвертации в соседнем текстовом поле.
5.Нажмите кнопку "Рассчитать", чтобы начать процесс конвертации.
6.В окне результатов будет отображен курс конверсии и сконвертированная сумма.
7.Закройте окно результатов, чтобы выполнить дополнительные конверсии.
8.Вы можете повторить процесс, выбрав другие криптовалюты или изменяя количество.

Интеграция с API
Приложение интегрируется с API CoinRanking для получения данных о криптовалютах в реальном времени.
Оно получает информацию о 50 самых популярных монетах на основе рыночной капитализации и обновляет 
локальную базу данных соответствующим образом.

База данных
Приложение использует MySQL в качестве системы управления базами данных для хранения информации о 
криптовалютах. Оно создает базу данных с именем currency и таблицу coins для хранения соответствующих данных.

Зависимости
Gson - Библиотека Java для сериализации и десериализации JSON.
MySQL Connector/J - Официальный JDBC-драйвер для MySQL.

4.Задайте соответствующие значения констант DB_URL, DB_USER и DB_PASSWORD.
